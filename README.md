## Wiring

### DHT22

First Pin is a VCC connect it to 5V.

Second Pin is Data Pin Connect it to Digital Pin 7.

Third Pin is called NC but i don't know why it's their i did not used it.

And Finally Fourth Pin is Ground Pin.
